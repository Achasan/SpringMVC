# SpringMVC : 백문이 불여일타

## 강의를 통해 배운 내용

- Servlet
  - Request, Response
    - HttpServletRequest 객체 기본 사용방법
    - HttpServletResponse 객체 기본 사용방법
    - Http 요청 데이터
      - GET (쿼리파라미터)
      - POST (HTML Form)
      - HTTP API(Json,XML ...)
